## 📚 新手必备资源库

这里精选了几个适合新手的入门级项目，每一个都亲测“友好”：

### 1. 🐍 Python 入门 · 小练习
- **资源类型**：交互式练习
- **推荐理由**：直接在浏览器里写代码，无需配置环境，30分钟上手基础语法
- **适合谁**：零基础，想先体验编程感觉的朋友
- **👉 快速体验**：[点击进入 Python 练习场](https://www.learnpython.org/)

### 2. 🌐 前端三件套 · 极简模板
- **资源类型**：可直接运行的 HTML/CSS/JS 模板
- **推荐理由**：看完就能用，改改文字就能做出自己的第一个网页
- **适合谁**：想学前端，但不想一上来就看枯燥文档的朋友
- **📁 仓库内直达**：点击查看 `/examples/first-website` 文件夹

### 3. 🎮 Git 可视化 · 游戏式学习
- **资源类型**：开源游戏
- **推荐理由**：用闯关的方式学 Git 命令，比看书有趣100倍
- **适合谁**：对 Git/GitHub 还不太熟悉，想要边玩边学的朋友
- **🎯 项目地址**：[Learning Git Branching](https://learngitbranching.js.org/?locale=zh_CN)

### 4. 📖 编程思维 · 免费电子书
- **资源类型**：中文翻译的开源书籍
- **推荐理由**：不讲具体语言，专门讲“怎么像程序员一样思考”
- **适合谁**：已经会写几行代码，但遇到问题不知如何下手的阶段
- **📘 阅读地址**：[笨办法学编程（思维篇）](https://www.bookstack.cn/read/think-like-a-programmer/README.md)
